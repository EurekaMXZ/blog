# blog

Personal blog.

## Development

```shell
git clone https://github.com/EurekaMXZ/blog.git
hugo mod tidy
hugo serve -D
```

## Deploy

Currently, [Vercel](https://vercel.com/) deployment is configured, fork and import to vercel, you'll be able to deploy your own site.
